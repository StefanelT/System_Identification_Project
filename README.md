# System_Identification_Project
📘 Project Overview

This project focuses on system identification techniques used to model dynamic systems based on experimental data. The main objective is to determine mathematical models for second-order systems with complex conjugate poles, using multiple identification approaches.

The project explores: 
• Identification using linear regression based on step response analysis 
• Identification using chirp signals in the frequency domain 
• Application of parametric methods such as ARMAX, OE, and state-space models

Experimental data is obtained through simulations and processed to estimate system parameters, validate models, and compare performance across different methods. ￼

⸻

📊 Key Features • Modeling of dynamic systems using real/simulated data • Signal preprocessing and noise filtering techniques • Parameter estimation (gain, damping factor, natural frequency) • Model validation using error metrics • Comparative analysis of different identification methods
